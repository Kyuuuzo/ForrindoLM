## ğŸ“¥ Cara Instalasi

Pastikan untuk membaca ini agar mod dapat berjalan dengan lancar:

1. Unduh dan install [SMAPI](https://smapi.io/) (Sangat disarankan untuk versi terbaru).
2. Unduh dan install [Content Patcher](https://www.nexusmods.com/stardewvalley/mods/1915).
3. Masuk ke halaman **Releases** di sebelah kanan halaman ini (atau klik [di sini](MASUKKAN_LINK_RELEASE_KAMU_NANTI_DISINI)).
4. Unduh file `[CP] Forrindo.zip` dari versi Alpha yang disediakan.
5. Ekstrak (Unzip) file tersebut ke dalam folder `Stardew Valley/Mods` pada komputer.
6. Jalankan game menggunakan `StardewModdingAPI.exe`.
